PERGUNTAS:
É o componente principal de um computador. Ela é responsável por executar instruções, realizar cálculos e coordenar as operações do sistema. É o cérebro do computador, processando dados e controlando outros componentes.
É um componente da CPU responsável por realizar operações lógicas (como comparações e operações booleanas) e operações aritméticas (como adição, subtração, multiplicação e divisão).
São componentes localizados dentro da CPU de um computador. Eles são usados para armazenar dados temporários e são extremamente rápidos em comparação com a memória principal.
 É uma memória volátil e de acesso aleatório, utilizada para armazenar temporariamente dados e instruções durante a execução de programas. É rápida e permite acesso rápido aos dados, mas perde as informações quando o computador é desligado.
É uma memória somente leitura, não volátil, que contém dados permanentes e instruções pré-gravadas. O conteúdo é definido durante a fabricação e não pode ser alterado pelo usuário.
É uma memória regravável do tipo ROM. Ela permite que os dados sejam apagados e regravados eletricamente, utilizando uma luz ultravioleta especial para apagar o conteúdo.
É uma memória não volátil e regravável eletricamente. Ela é amplamente utilizada em dispositivos como pen drives, cartões de memória, smartphones e SSDs (Solid State Drives).É mais lenta do que a RAM, mas oferece capacidade de armazenamento permanente e maior durabilidade.
É um tipo de memória usada para armazenar grandes volumes de dados de forma permanente, mesmo quando o computador é desligado. Exemplos incluem discos rígidos (HDDs), unidades de estado sólido (SSDs), discos ópticos (CDs, DVDs) e fitas magnéticas. É mais lenta em comparação com a RAM, mas oferece capacidade de armazenamento em larga escala.
É uma técnica que permite a transferência direta de dados entre dispositivos periféricos e a memória, sem a intervenção direta da CPU. É uma forma eficiente de mover dados rapidamente, reduzindo a carga de trabalho da CPU.
 É um sinal utilizado para selecionar um dispositivo específico em um barramento ou conjunto de dispositivos. Ele indica qual dispositivo deve responder às comunicações e evita conflitos entre dispositivos. É comumente usado em interfaces de comunicação, permitindo a transferência correta de dados para o dispositivo selecionado.
 Transmite os endereços de memória ou dispositivos, indica onde os dados estão localizados.
Transmite os dados em si entre a CPU, a memória e os dispositivos periféricos, transportando-os.
Oferece um bom equilíbrio entre desempenho e preço. Geralmente possui quatro núcleos, cache inteligente e frequências de clock variáveis. Alguns modelos suportam Hyper-Threading para multitarefa. O TDP é moderado, resultando em menor consumo de energia.
 É conhecido por alto desempenho em computadores de alto padrão. Possui quatro, seis ou oito núcleos, suporte a Hyper-Threading, cache avançado e frequências de clock mais altas. TDP geralmente é maior que o i5 devido ao maior número de núcleos e frequências mais altas.
 Possui dois núcleos de processamento.
Possui quatro núcleos de processamento.
São unidades de execução dentro de um programa que permitem a realização simultânea de tarefas. Elas são usadas para executar partes diferentes do programa de forma concorrente, aproveitando melhor os recursos do processador.
É utilizada para armazenar temporariamente dados frequentemente acessados pela CPU, reduzindo o tempo de acesso à memória principal. Ela melhora o desempenho do processador ao fornecer um acesso mais rápido aos dados, diminuindo a latência.
